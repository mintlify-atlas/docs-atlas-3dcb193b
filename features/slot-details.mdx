The Slot Details page provides comprehensive analysis of individual slot performance, including transaction breakdowns, compute unit utilization, processing timelines, and detailed execution metrics.

## Page Access

The Slot Details route (`src/routes/slotDetails.tsx`) accepts a slot number as a URL parameter:

```tsx
const searchParamsSchema = z.object({
  slot: z.number().optional().catch(undefined),
});

export const Route = createFileRoute("/slotDetails")({
  validateSearch: searchParamsSchema,
  component: SlotDetails,
  beforeLoad: ({ search: { slot } }) =>
    store.set(baseSelectedSlotAtoms.slot, slot),
});
```

### Navigation to Slot Details

You can access slot details in multiple ways:

1. Click any slot in the [Overview slot timeline](/features/overview#slot-timeline)
2. Click a slot card in the [Leader Schedule](/features/leader-schedule)
3. Direct URL navigation: `/slotDetails?slot=12345`
4. Use the slot search feature

## Main Component

The Slot Details feature (`src/features/SlotDetails/index.tsx`) handles slot loading and validation:

```tsx
export default function SlotDetails() {
  const selectedSlot = useAtomValue(selectedSlotAtom);
  const [waitedForDependencies, setWaitedForDependencies] = useState(false);
  const isNotReady =
    useAtomValue(baseSelectedSlotAtoms.state) ===
    SelectedSlotValidityState.NotReady;

  useEffect(() => {
    if (isNotReady && !waitedForDependencies) {
      const timeout = setTimeout(() => setWaitedForDependencies(true), 2_500);
      return () => clearTimeout(timeout);
    }
  }, [isNotReady, setWaitedForDependencies, waitedForDependencies]);

  // Wait for dependencies before showing error
  if (isNotReady && !waitedForDependencies) return null;

  return selectedSlot === undefined ? <SlotSearch /> : <SlotContent />;
}

function SlotContent() {
  const slot = useAtomValue(selectedSlotAtom);
  if (slot === undefined) return;

  return (
    <Flex direction="column" gap="2" flexGrow="1">
      <SlotNavigation />
      <ChartControlsProvider>
        <DetailedSlotStats />
        <SlotPerformance />
        <ComputeUnitsCard />
        <TransactionBarsCard />
      </ChartControlsProvider>
    </Flex>
  );
}
```

| **CU Limit** | Maximum CUs available per block |
| **Utilization %** | Percentage of available CUs used |
| **Average CU per TX** | Mean compute units per transaction |

## Troubleshooting

### Slot Data Not Loading

1. Verify slot number is valid (within current epoch range)
2. Check that slot has been processed by the validator
3. Ensure sufficient time has passed for data collection
4. Verify API connectivity

### Missing Transaction Data

Some transactions may not have complete data if:
- Slot was skipped
- Data retention period expired
- Transaction failed early in execution

### Chart Performance Issues

If experiencing slow chart rendering:
1. Try filtering transaction list
2. Use the transaction bars view instead of detailed chart
3. Reduce the number of visible annotations
4. Close other browser tabs to free memory
