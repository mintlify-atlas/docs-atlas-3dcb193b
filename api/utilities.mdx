## General Utilities (utils.ts)

### getLeaderSlots

Gets all leader slots for a specific validator in an epoch.

```tsx
function getLeaderSlots(epoch: Epoch, pubkey: string): number[]
```

**Parameters**

<ParamField path="epoch" type="Epoch" required>
  The epoch object containing leader schedule data
</ParamField>

<ParamField path="pubkey" type="string" required>
  The validator's public key
</ParamField>

**Returns** Array of slot numbers where the validator is the leader

```tsx
import { getLeaderSlots } from "./utils";

const mySlots = getLeaderSlots(currentEpoch, myPubkey);
console.log(`I'm leader for ${mySlots.length} slots`);
```

### getSlotGroupLeader

### getDurationText

Formats a Luxon Duration into human-readable text.

```tsx
function getDurationText(
  duration?: Duration,
  options?: DurationOptions
): string
```

**Options**

<ParamField path="showOnlyTwoSignificantUnits" type="boolean">
  Limit output to two most significant time units
</ParamField>

<ParamField path="omitSeconds" type="boolean">
  Don't include seconds in output
</ParamField>

```tsx
import { getDurationText } from "./utils";
import { Duration } from "luxon";

const duration = Duration.fromObject({ hours: 2, minutes: 30, seconds: 45 });
console.log(getDurationText(duration)); // "2h 30m 45s"
console.log(getDurationText(duration, { omitSeconds: true })); // "2h 30m"
console.log(getDurationText(duration, { showOnlyTwoSignificantUnits: true })); // "2h 30m"
```

## Number Utilities (numUtils.ts)

### formatNumber

Formats numbers with fixed decimals or significant digits.

```tsx
type FormatNumberOptions =
  | {
      decimals: number | ((value: number) => number);
      trailingZeroes?: boolean;
      decimalsOnZero?: boolean;
      useSuffix?: boolean;
    }
  | {
      significantDigits: number;
      exactIntegers?: boolean;
      trailingZeroes?: boolean;
      decimalsOnZero?: boolean;
      useSuffix?: boolean;
    };

function formatNumber(value: number, opts: FormatNumberOptions): string
```

**Examples**

```tsx
import { formatNumber } from "./numUtils";

// Fixed decimals
formatNumber(1234.5678, { decimals: 2 }); // "1,234.57"

// Significant digits
formatNumber(0.00012345, { significantDigits: 3 }); // "0.000123"

// With suffix
formatNumber(1_500_000, { decimals: 1, useSuffix: true }); // "1.5M"

// Dynamic decimals
formatNumber(123.456, {
  decimals: (val) => (val < 1 ? 4 : 2),
}); // "123.46"
```

## Transaction Utilities (transactionUtils.ts)

### getTxnStateDurations

Calculates durations for different transaction execution phases.

```tsx
function getTxnStateDurations(
  transactions: SlotTransactions,
  txnIdx: number,
  bundleTxnIdx: number[] | undefined,
  client: Client
): {
  preLoading: bigint;
  validating: bigint;
  loading: bigint;
  execute: bigint;
  postExecute: bigint;
}
```

**Returns** Object with phase durations in nanoseconds

### getTxnState

Determines the current execution state of a transaction at a given timestamp.

```tsx
function getTxnState(
  ts: number,
  transactions: SlotTransactions,
  txnIdx: number,
  bundleTxnIdx: number[] | undefined,
  client: Client
): TxnState
```

**Returns** One of: `PRELOADING`, `VALIDATE`, `LOADING`, `EXECUTE`, `POST_EXECUTE`

### getTxnBundleStats

Gets statistics about a transaction bundle.

```tsx
function getTxnBundleStats(
  transactions: SlotTransactions,
  txnIdx: number
): TxnBundleStats
```

**Returns**

- `totalCount` - Total transactions in bundle
- `order` - Position of this transaction in bundle
- `bundleTxnIdx` - Array of all transaction indices in bundle

## Clock Utilities (clockUtils.ts)

### createClock

Creates a requestAnimationFrame-based clock for synchronized updates.

```tsx
function createClock(intervalMs: number): {
  subscribeClock: (fn: (now: number, dt: number) => void) => () => void;
}
```

**Example**

```tsx
import { createClock } from "./clockUtils";

const clock = createClock(16); // ~60fps

const unsubscribe = clock.subscribeClock((now, dt) => {
  console.log(`Frame at ${now}ms, delta: ${dt}ms`);
});

// Later...
unsubscribe();
```
